# Print Cost Comparator

A web application designed to compare printing costs across different copy shops based on uploaded PDF documents.

## About the Project

This project started from a simple real-world problem: calculating how much it would cost to print a PDF document in different copy shops from my city.

The application analyzes uploaded PDF files, calculates the number of pages, and estimates printing costs based on each copy shop pricing configuration.

Beyond the technical side, this project helped me understand how to transform an idea into a functional product focused on solving a real user need.

## Features

- PDF upload and processing
- Automatic page count calculation
- Printing cost estimation
- Comparison between different copy shops
- Simple and practical user experience

## Tech Stack

- HTML
- CSS
- JavaScript
- Python Flask

This was one of my first end-to-end projects and played an important role in my learning journey as a developer. It helped me better understand product thinking, problem solving, and how to build complete web applications around real-world needs.
